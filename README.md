# dyno-jet-log-table-generator-app
This is a quikc applicatin I designed to read data from a DynoJet tune log file.  You can then pigeon-hole the data onto an X Y axis.  For example 1389 RPM and 43.86 MAP readings do not have a table cell in a tune table; You have to decide if that data point gets mapped to 1250 or 1500 RPM and 40 or 45 MAP.
